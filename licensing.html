Licensing information

The basic license of our computational models includes a minimum of two protection dongles, which allows running our software on up to two different computers simultaneously, although there is no limitation on the number of installations of our software on different physical machines. PSR dongles are only needed to run the models, so many people from the team may install the tool on their own PCs and open the interfaces and graph results through.

Annual Subscription

In this type of licensing agreement the user pays a recurring fee each year to access and use the software. During this period, the user has access to the updates, maintenance and support (M&S) service. If the customer chooses not to renew the subscription in the second year, the customer will no longer be able to run the software.

Perpetual License

In this type of licensing agreement the user a specific version of the software indefinitely and also has access to the updates, maintenance and support (M&S) service for the period of 1 year. After this period, to be entitled to receive support, maintenance, and model updates, the Customer must contract the respective maintenance and support (M&S) service.

Leasing

In this type of licensing agreement the user pays to use the software for a specified period, typically a 3-month period, with the option to renew or return the software at the end of the lease term.
