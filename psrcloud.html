PSR Cloud is a cutting-edge platform designed to manage remote model execution in a distributed processing environment. With this platform, users can achieve high-performance computing without the need for local infrastructure. PSR Cloud features a powerful network equipped with thousands of processors, ensuring efficient and scalable processing capabilities.

High-performance

Cost reduction

Reduces the investment in dedicated hardware, software, and personnel

Scalability

Allows the execution of multiple cases simultaneously

Confidentiality and Security

All data communication (input and output files of the models) is carried out, using data cryptography, directly between the user's local computer and the remote servers allocated to the execution, without any interference or passing through PSR servers. There is no access to the user's data by third parties.

PSR Cloud Journey

The user informs the study input data folder and the desired number of processors.

When the user executes the case, the PSR Cloud Server verifies the permissions granted to the user and allocates the requested servers.

PSR Cloud Client sends to the allocated serves the models' input data. Once the execution is finished, the results can be transferred to the user's local computer.

PSR Cloud is a pay-per-run service for SDDP, OptGen and NCP

PSR Cloud is billed as a pay-per-run service on a prepaid basis. In prepaid mode, the user acquires credits to be used in executions. The cost of each round is accounted for and deducted from the current balance.
